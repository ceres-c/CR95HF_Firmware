int8_t ISO15693_GetTagIdentification ( uint16_t *  Length_Memory_TAG,
uint8_t *  Tag_Density,
uint8_t *  IC_Ref_TAG 
)

Find the TAG ISO15693 present in the field.

Parameters
*Length_Memory_TAG: return the memory size of the TAG
*Tag_Density: return 0 if the TAG is low density (< 16kbits), return 1 if the TAG is high density
*IC_Ref_TAG: return the IC_Ref of the TAG ( the 2 LSB bits are mask for the TAG LRiS2K, LRi2K and LRi1K)
Return values
ISO15693_ERRORCODE_DEFAULT/ ISO15693_SUCCESSCODE.

uc8  Length,
uint8_t *  Flags,

uint8_t *  UIDoutIndex 
)

this function splits inventory response. If the residue of tag response is incorrect the function returns ERRORCODE_GENERIC, otherwise ISO15693_SUCCESSCODE

Parameters
ReaderResponse: pointer on PCD response
Length: Number of byte of Reader Response
Flags: Response flags
DSFIDextractDSPID of tag response
UIDoutIndex: index of UIDout of tag response
Return values
ISO15693_SUCCESSCODE: Contactless tag response validated
CR95HF_ERROR_CODE: PCD teturned an error code
ISO15693_ERRORCODE_CRCRESIDUE: CRC16 residue is erroneous
