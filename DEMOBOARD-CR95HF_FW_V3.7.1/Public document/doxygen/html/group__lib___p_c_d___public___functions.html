int8_t PCD_CheckSendReceive ( uc8 *  pCommand,
uint8_t *  pResponse 
)

this function send a command to the PCD device device over SPI or UART bus and receive its response.

the returned value is PCD_SUCCESSCODE

Parameters
*pCommand: pointer on the buffer to send to the the PCD device ( Command | Length | Data)
*pResponse: pointer on the the PCD device response ( Command | Length | Data)
Returns
PCD_SUCCESSCODE : the the PCD device returns an success code
PCD_ERRORCODE_DEFAULT : the the PCD device returns an error code
